eXplainable PRS (XPRS)

Case 1: You have a cohort genotype file

If you have the entire cohort genotype file and want to start running the platform for the first time, go to Case 1.

Case 1-1: You have a cohort genotype file & RDS file

If you have the entire cohort genotype file and have already run the platform to obtain the RDS file (result file from the first run from the Case 1), go to Case 1-1.

Case 2: You have a reference genotype file & individual genotype file

If you have a reference genotype file and an individual genotype file, and want to analyze the individual genotype file using the reference, follow this process:

  • First, upload the reference genotype file. You can download reference genotype files from sources like 1000G or UKB.
  • Then, upload the individual genotype file then get analysis.

To start this process, go to Case 2.

Case 2-1: You have a reference genotype file, individual genotype file & RDS file

If you have the reference genotype file, individual genotype file, and an RDS file (result file from the first run from Case 2), go to Case 2-1 for the further analysis for individual.
